## 1. Background

AdamW는 현대 딥러닝에서 가장 널리 사용되는 옵티마이저 중 하나입니다. 업데이트 규칙은 다음과 같습니다:

$$\theta_{t+1} = \theta_t - \alpha_t \frac{\hat{m}_t}{\sqrt{\hat{v}_t} + \epsilon} - \alpha_t \lambda \theta_t$$

여기서 $\hat{m}_t$와 $\hat{v}_t$는 각각 bias-corrected 1차, 2차 모멘트 추정치입니다:

$$\hat{m}_t = \frac{m_t}{1 - \beta_1^t}, \quad \hat{v}_t = \frac{v_t}{1 - \beta_2^t}$$

마지막 항 $-\alpha_t \lambda \theta_t$가 L2 regularization이 아닌 **decoupled weight decay**입니다.

## 2. Martingale Formulation

확률적 그래디언트 노이즈를 $\xi_t = \nabla f(\theta_t) - g_t$라 정의하면 (여기서 $g_t$는 미니배치 그래디언트), $\{\xi_t\}$는 자연스러운 filtration $\mathcal{F}_t$에 대한 martingale difference sequence를 이룹니다:

$$\mathbb{E}[\xi_t \mid \mathcal{F}_{t-1}] = 0$$

Lyapunov function $V_t = f(\theta_t) + \frac{C}{\alpha_t}$를 구성하면:

$$\mathbb{E}[V_{t+1} \mid \mathcal{F}_t] \leq V_t - \frac{\alpha_t}{2}\|\nabla f(\theta_t)\|^2 + \frac{L \alpha_t^2 \sigma^2}{2B}$$

## 3. Main Theorem

**Theorem 1.** $f$가 $L$-smooth이고 하한이 있으며, 그래디언트 노이즈 분산 $\sigma^2 < \infty$라 하자. 학습률 $\alpha_t = \alpha / \sqrt{t}$를 사용하면:

$$\frac{1}{T} \sum_{t=1}^{T} \mathbb{E}\!\left[\|\nabla f(\theta_t)\|^2\right] \leq \mathcal{O}\!\left(\frac{1}{\sqrt{T}}\right) + \mathcal{O}\!\left(\frac{\sigma^2 \log T}{B\sqrt{T}}\right)$$

*Proof sketch.* Azuma-Hoeffding 부등식을 martingale difference $\xi_t$에 적용하고, stopping time $\tau = \inf\{t : \|\nabla f(\theta_t)\| \leq \varepsilon\}$에서의 기댓값을 bound합니다. $\square$

기존 Adam 분석([Défossez et al., 2022])에 비해 로그 인자를 추가로 포착합니다.

## 4. Implementation Note

```python
import torch

class AdamWAnalytic(torch.optim.AdamW):
    """AdamW with learning rate schedule matching Theorem 1."""

    def __init__(self, params, alpha0=1e-3, **kwargs):
        super().__init__(params, lr=alpha0, **kwargs)
        self.alpha0 = alpha0
        self._step  = 0

    @torch.no_grad()
    def step(self, closure=None):
        self._step += 1
        # α_t = α_0 / √t  (Theorem 1 schedule)
        lr_t = self.alpha0 / (self._step ** 0.5)
        for g in self.param_groups:
            g['lr'] = lr_t
        return super().step(closure)
```

## 5. Empirical Validation

GPT-2 small (124M)을 OpenWebText에서 학습한 결과, 이론적 수렴 곡선과 실험 결과가 잘 일치함을 확인했습니다. 특히 초기 학습 단계($t < 1000$)에서 예측된 bound의 tight함이 두드러집니다.
